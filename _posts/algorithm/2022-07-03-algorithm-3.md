### Scanner 관련
1. int 입력 받고 buffer 비우기
2. 구분자없이 연속으로 받을경우 String -> Character -> 아스키코드 '0' 빼기
```java
Scanner scanner = new Scanner(System.in);
N = scanner.nextInt();
M = scanner.nextInt();
scanner.nextLine();

drink = new int[N][M];

for (int i = 0; i < N; i++) {
    String str = scanner.nextLine();
for (int j = 0; j < M; j++) {
    drink[i][j] = str.charAt(j) - '0';
}
```
